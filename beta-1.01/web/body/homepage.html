The hardest thing to think was the "why". "Why am' I creating this website, or even why I decided to do that".

Well... I think it's because, initially, I really thought it would be a good idea to start my own business from scratch, without taking any money out of my pocket, or even if I had to, as little as possible.

Everything startded when I was playing minecraft with some of my friends and we were having some troubles with the many versions I was creating for the same modpack and with the constant updates I was doing in the server.

So I thought I'd create a website to document all the changes to the Minecraft server and modpacks, as well as organizing all this documentation.

As well as storing everything related to the game servers we used to play, like a backup of them in case we want to play them again, even though we'll probably never play them again. But it's there as a souvenir.

But this ideia endup been a website portfolio for my future projects.
